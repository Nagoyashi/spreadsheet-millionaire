# v0.8.0 — Launch: two-environment cutover (2026-06-14)

The production launch. www.spreadsheetmillionaire.com now serves `main`.

- **Env-driven API proxy:** a zero-dependency Vercel Edge Middleware reads
  `BACKEND_ORIGIN` at the edge, so production and preview deployments proxy to
  separate backends and the backend URL never enters the client bundle (#49).
- **Two-environment topology:** production (Vercel + Render on `main`, Neon
  production branch) and staging (a second Render service on `develop`, Neon dev
  branch); `BACKEND_ORIGIN` scoped per Vercel environment.
- **Verified live:** environment isolation both directions; production auth
  (register / login / save / password-reset) confirmed.
