# v0.13.2 — Marketing landing polish (2026-06-30)

Visual polish of the public marketing page, from design feedback.

## Header
- The nav now spans the **full page width** — wordmark hard-left, the "Login App"
  control hard-right — instead of being boxed into a centered column.

## Carousels
- New `Carousel`: a **continuous, one-card-at-a-time** slider. It shows a full
  page of cards (responsive 1 / 2 / 3 / 4) and shifts by a single card — the next
  slides in from the right, the leftmost slides out — looping seamlessly. Autoplay
  (pause on hover) plus manual side arrows. No more cut-off cards.
- **"Calculators you can use right now"** uses it, so the row scrolls as more
  calculators are published instead of wrapping onto new lines.
- **"More on the way"** uses it too, and now shows the **whole roadmap**: the two
  trackers, the not-yet-published calculators (everything the runtime publish set
  omits), and the Guide / Comparison / ETFs & Stocks sections — each upcoming-page
  card linking to its coming-soon page.

## Sections
- The **"Why SpreadsheetMillionaire"** value props are now boxed cards with a top
  border, so the section reads as distinct from the calculator showcase above.

Frontend-only; no backend or schema changes.
