# v0.13.1 — Marketing header + section placeholders (2026-06-29)

A marketing-site patch: rework the public header and stand up the first
navigation to the upcoming content/affiliate/search surfaces.

## New header layout
The marketing nav is now a three-column layout:
- **Left** — the `SpreadsheetMillionaire` wordmark (links home).
- **Center** — the section links: **Guide · Calculators · Comparison · ETFs and Stocks**.
- **Right** — **Login App**: a non-clickable control that reveals **Login** and
  **Register** on hover (or keyboard focus). An already-authenticated visitor sees
  **Open app** instead. Mobile collapses everything into the disclosure menu.

## Section routing
- **Calculators** → the calculators app (`/app`).
- **Guide**, **Comparison**, **ETFs and Stocks** → Beta **"coming soon"** pages
  (`/guide`, `/comparison`, `/etfs-stocks`) — one shared `MarketingComingSoonPage`
  with a per-section title/blurb, on the same stone canvas as the rest of the site.

These three sections are their own later release cycles (noted in `project.md`
§ Future):
- **Guide** — the resource center / SEO article hub (organic-growth engine).
- **Comparison** — broker/account comparisons with affiliate links (income stream).
- **ETFs and Stocks** — real-time, categorized ETF/stock search.

No backend or schema changes. Tests: a new `MarketingNav` spec (sections, the
non-clickable Login App + its Login/Register links, the authed "Open app" state);
frontend suite green.
